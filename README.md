## 🎯 Visión General

Routime es una app para entrenar en casa con rutinas tipo HIIT, Tabata o circuitos. Crea ejercicios, arma rutinas y ejecútalas con temporizador preciso y feedback de voz, todo offline.

## ✨ Características

- Crear, editar y eliminar ejercicios con video
- Crear, editar y ejecutar rutinas con intervalos personalizables
- Temporizador preciso y controles de ejecución
- Síntesis de voz integrada
- Almacenamiento local en IndexedDB
- Tema claro/oscuro

## 🚀 Demo rápida

1. Crear un ejercicio con video.
2. Crear una rutina con intervalos.
3. Ejecutar la rutina y seguir el temporizador con voz.

## 🛠️ Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Zustand
- Dexie (IndexedDB)

## 📦 Instalación
